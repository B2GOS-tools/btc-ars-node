# Bitcoin en Argentina

Built with the following front-end libraries:

* [Ratchet][rtch]
* [localForage][fora]
* [numeral][nume]
* [moment][mome]
* [zepto][zpto]

and back-end Node packages:

* [hapi.js](hapi)
* [slm-lang](slm)

## Bootstrap

    $ git submodule add https://vivaserver:bitargon@bitbucket.org/vivaserver/btcbitsapp.git public

    $ npm install
    ...
    $ foreman start
or
    $ node .
