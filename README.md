# Bitcoin en Argentina Node.js backend

NPM package dependencies:

* [hapi.js][hapi]
* [slm-lang][slml]

## Bootstrap

    $ git submodule init
    $ git submodule update

    $ npm install
    ...
    $ foreman start
or
    $ node .
