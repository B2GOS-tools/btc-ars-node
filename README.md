# Bitcoin en Argentina

Built with the following front-end libraries:

* [Ratchet][rtch]
* [localForage][fora]
* [numeral][nume]
* [moment][mome]
* [zepto][zpto]

## Bootstrap

    $ npm install
    ...
    $ foreman start
or
    $ node .
