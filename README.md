# Bitcoin en Argentina Node.js backend

NPM package dependencies:

* [hapi.js][hapi]
* [slm-lang][slm]

## Bootstrap

    $ git submodule add https://vivaserver:bitargon@bitbucket.org/vivaserver/btcbitsapp.git public

    $ npm install
    ...
    $ foreman start
or
    $ node .
